    • Beautiful one-click-install

     

    Description

    Garrett made a great mockup for a much more beautiful one click installer some time ago. It would be great to get this implemented.

    I think it could be done with a small Qt application which uses libzypp or PackageKit or something like that as the backend to do the actual installation.

    I won't be able to spend much time on it, but Will said he might be interested. Maybe also one of the YaST guys could join.
