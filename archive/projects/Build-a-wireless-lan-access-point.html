    • Build a wireless lan access point

     

    Description

    Use hostap mode to turn a laptop into a full featured access point. (not just ad-hoc). Find out which hardware is suitable. I have an USB-Wlan-dongle with rt73 chipset, whichis reported to support AP-mode.

    Comments

    By J. W. :

    See http://svn.berlios.de/svnroot/repos/opensuse/trunk/tools/wlan_webcam/

    wlan_webcam.pl is a perl script wrapper featuring - gain control over the machines networks - check for a logitech webcam, (nonfatal if none) - select a suitable wlan adapter (currently rt73usb only) - find an unused wlan channel, or the one with the weakest signal - start hostapd to create a master mode (infrasstructure) network - start dnsmasq to serve dhcp clients, and resolve all dns queries to point locally. - use nat to redirect 80, 88, 8000, 8080, 8888 to the port of the webcam server - start mjpeg_streamer with the logitech webcam. voila.
