    • Get Enlightenment 17 into openSUSE

    Description

    Enlightenment 17 is good looking, but resource friendly, very well usable desktop environment. Unfortunately it's not present in openSUSE distribution and current state of X11:Enlightenment is awful. As a result, there were several home projects with these packages and no effort to fix current state.

    Project X11:Enlightenment:Nightly is following SVN revisions closely and is already providing well working packages for a while. To improve this repository we need to add more packages

    We also have X11:Enlightenment:Factory which contains released versions of EFL and E17 and packages are slowly accepted from this project to openSUSE:Factory.

    More things to do:

    • It would be great to integrate E17 to openSUSE installer so it would be an option for desktop environment choice.
    • create another project containing not released (but useful software) built against released EFL and E17 versions
    • test and report bugs
