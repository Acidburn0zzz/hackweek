    • Integrate Chubby-like lock service into SLES

    Description

    This is a continued work of the feature in Hackweek V: Implement Chubby-style lock service for cloud computing with SLEHA cluster stack (https://features.opensuse.org/309628). In this hackweek, I'd like to integrate this Chubby-like lock service into SLES, to see how it performed and what needs to be improved.

    Future work might be the comparison between Paxos implementation, Corosync and BDB's internal replication function, and to see which approach is the best way to resolve the consensus problem in the cloud environment.
