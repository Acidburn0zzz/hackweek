    • Libvirt for android

     

    Description

    Because the android'toolchain doesn't support glibc environment and it is based on ARM platform, The project needs to change the libvirt code a little bit, recompile libvirt with android toolchain,Make it available for android 2.2 system.

    Based on this new libvirt, you can create some management tools of virtualization and deploy them to your mobile devices and perform some management tasks,just like what you do on pc.
