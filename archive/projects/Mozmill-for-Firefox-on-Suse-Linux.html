    • Mozmill for Firefox on Suse Linux

     

    Description

    Mozmill is a open-source Automation test tools for firefox which developed by Mozilla, We want to introduce it into Suse linux test workflow on firefox.

    Not all of the test cases in Mozmill are suit for firefox version on Suse linux, So a certain of changes will be optimized and improved.
