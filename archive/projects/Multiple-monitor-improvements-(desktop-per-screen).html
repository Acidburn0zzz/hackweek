    • Multiple monitor improvements (desktop per screen)

    Description

    The aim of this hackweek project is to improve work with multiple monitor setups. The main two areas are:

    - Fix problems that show in such setup. - Make it possible to have one virtual desktop assigned to one screen, instead of the current way of having each virtual desktop span all screens. This will make screens more independent and hopefully easier to use.
