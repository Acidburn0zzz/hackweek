    • Support Remote Control for Pentax K series DSLR cameras

    Description

    libgphoto2 does not currently have support for remote controlling Pentax K DSLRs. There are however standalone projects, like pkremote and pktriggercord that implement it.

    The protocol is yet again special SCSI command tunneling over USB Mass Storage.
