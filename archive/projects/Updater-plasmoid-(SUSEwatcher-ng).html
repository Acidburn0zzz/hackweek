    • Updater plasmoid (SUSEwatcher-ng)

    Description

    Write an updater plasmoid modeled after SUSEwatcher from the good old days.

    That means a simple applet, which uses libzypp to check for patches, notifies the user about them, and launches YOU to install them.

    Such a simple design should be relatively low-maintenance, and not very prone to strange errors and behaviour - unlike packagekit based updaters.
