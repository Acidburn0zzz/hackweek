    • A generic HRTF library for virtual surround

    Tags: Idea, Desktop, sound

    Description

    The virtual surround is used in wide ranges of desktop systems -- except for Linux :) The typical implementation of virtual surround is HRTF (Head-Related Transfer Functions) or HRIR (Head-Related Impulse Response), by setting the audio source in 3D position and spacializing them.

    There are some reference implementations and freely available HRTF data. So, the materials and recipes are there...

    People

    Takashi Iwai originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
