    • Add a P2P TV watching application

    Tags: Idea, multimedia

    Description

    There's an P2P application applied to Windows through which users can watch TV online and real-time which is called pplive. Opcast is something similar which can be applied to Linux but with less channels. Maybe we can add this application to our StartMenu -> Multimedia -> TV-online. Users don't need a TV card hardware any more.

    People

    Yi Xu originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
