    • Add game that can be played during installation

    Tags: InProgress, openSUSE, games, fun

    Description

    Back in the Caldera days, the user got the option to play tetris while installation was in progress. I always liked this feature and think it's much more fun than watching the slide show or the details view.

    We could use a different game for each version of openSUSE.

    I talked to YaST people about this and I was told that technically it's not hard to do (I guess every QT based (or even KDE?) game will work) but it was never accepted. For openSUSE I think this would be great ... for SLE* it's probably not a good idea ;-)

    People

    Daniel Bornkessel originated this idea.
