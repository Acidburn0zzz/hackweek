    • Add Twitter support to Banter

    Tags: Idea, desktop, twitter, banter

    Description

    Add a Twitter plugin for Banter so a user can easily twitter from within a Banter chat window. Two possible implementations the first a full blown telepathy connection manager interface on top of a twitter client. The second just add a twitter module to Banter so a user can type "/twitter Hello World" from the chat window to actually twitter. Friend twits would appear as a twit message in the chat window.

    People

    Brady Anderson originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
