    • Apple's Automator "clone"

    Tags: InProgress, Desktop, UnixFeature

    Description

    Apple's Automator provides a very good way for any kind of users to write scripts or just automate common tasks. We could do something similar, by defining a common way for applications to install "actions" and then, using that, we could have GNOME/KDE frontends, both using the same infrastructure.

    People

    Rodrigo Moya originated this idea. There is some code in GNOME SVN (see related materials)

    Related Materials

    A very basic start for this is in http://svn.gnome.org/viewcvs/atomato/ http://www.apple.com/macosx/features/automator/
