    • Autoshrink a certified distro from SUSE to a micro-image - for dummies - and KISS

    Tags: Management, Virtualization, Server, SLES, SLED, openSUSE, SLEPOS, InProgress, HelpWanted, HW-Hacker

    Description

    o Main Goal:

    Install a machine with a certified set of packages and automatically generate from that a micro image and a couple of (definable) addon overlays that you then can easily stack together, image and deploy on virtual instances, or use for a rescue system sitting on a PXE server, USB stick or local harddisk partition or image.

    o Sub Goals:

    o Stay as close to the certified original as possible (like starting from a SLES 10 SP1 minimal install, which is still too big for my target of a small but stackable micro+addon toolkit) o Keep at least the MD5SUM on a file level intact to the certified original o Easy to use this auto-shrinker => a script or fully automate it o Easy to develop it => My programming skills are moderate and becoming a manager did not improve them either ;-) But if you give me a pool of RPMs and a UNIX command line I can do interesting stuff like LISA (Caldera's text based installer) ;-) o Contribute something that might be usefull for our customers and field engineers, work with others in a team during hackweek who have similar/complementary ideas/skills and then merge projects and create a UNIX-pipe-onliner-superthing by interfacing with their projects (pre- or afterburners). If somebody builds a nice GUI frontend and yet other write some documentation around it and its architecture, this would be my dream of how hackfest should work ;-) o KISS (those who not know what this means ask others and make friends by getting closer to the community, if all that fails ask me ;-)
