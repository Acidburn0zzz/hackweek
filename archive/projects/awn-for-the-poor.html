    • AWN for the poor

    Tags: Idea, Desktop, GNOME

    Description

    Create a dock or a dock-like thing for people, who don't have all that fancy Compiz bling-bling. Current alternatives I'm aware of either need compositing for their effects or are very resource hungry and not well integrated (gdesklets). Gimmie might already be too much.

    It would be nice to have similar feature of the dock (maybe even the new stacks in Leopard) without animations and effects. I think a lowtech-dock would be equally useful to a fancy dock.

    The most simple solution might be to tweak the current window list to show icons only and not using the button effect.

    People

    Joerg Kress originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
