    • B-Spot - EBooks Management Tool

    Tags: Idea, SLED, Desktop

    Description

    In the same way how F-Spot manages Fotos, we need a new application B-Spot that manages one's E-Books.

    Books should be labellable - say "Operating Systems", "Compilers", "Philosophy" etc.

    Contents of the books should be searchable via authors, contents, labels. Like, if I give a search-query as:

    label:Software "no silver bullet"

    then b-spot should return me with "Mythical Man Month".

    B-Spot should have a viewer that should be able to automatically view any file type viz: PDF, CHM, ODS, DOC, HTML etc

    Instead of trying to implement a search functionality by itself, B-Spot could make use of the services of Beagle.

    All E-Books should be saved to user's home directory.

    If there is a network available and if the user searches for a book, if it is not available with the user, it could make use of Amazon / Froogle APIs and suggest user where he can get them.

    Optionally, B-Spot can get reviews for books and associate them as metadata so that when an user searches a book, he get review-comments, errata, suggestions etc.
