    • browser-based shared whiteboard

    Tags: Web20, InProgress, Whiteboard

    Description

    Goal of this project is to create a web application that allows two or more users to draw and write stuff on a whiteboard-like page (a bit like http://www.skrbl.com/ but using open source technology instead of asp).

    Server:

    • one backend process managing the whiteboards and widgets on them
    • a set of http server processes using drb to get the whiteboards/widgets from the backend
    • http server used is thin (http://code.macournoyer.com/thin/)
