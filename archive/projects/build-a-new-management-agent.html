    • Build a new management agent

    Tags: Management, InProgress

    Lightning Talk

    Description

    Web Services for Management (WS-Man) is the new kid on the block in the area of cross platform management standards. It has quite a lot of industry support.

    WS-Man does not include an information model, so the need for the Common Information Model (CIM) still exists. In fact, most all efforts with WS-Man are using the WS-Man CIM bindings specification.

    Linux could benefit from a new systems management agent that * Supports WS-Man natively * Is a CIM Object Manager * Supports CIM-XML * Supports non-CIM plugins for WS-Man

    The time is right for such an agent to be written in a dynamic language such as Python.

    People

    Bart Whiteley originated this idea. Bart Whiteley and Jon Carey are working on it.
