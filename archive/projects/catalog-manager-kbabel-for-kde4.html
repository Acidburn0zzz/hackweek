    • Catalog Manager (KBabel) for KDE4/Qt4

    Tags: TryMe, Qt, KDE, i18n

    Description

    KBabel is a powerful translation application for PO files. Catalog manager is one part of it, displaying overview of those PO files with advanced features as search, replace or SVN/CVS repository access. The goal of this project is to rewrite Catalog Manager from scratch to keep it on par with Qt4 / KDE 4 foundation. The initial prototype will be written as a pure Qt4 app.
