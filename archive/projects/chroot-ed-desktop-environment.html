    • Chroot'ed desktop environment

    Tags: Security, Desktop

    Description

    Make desktop chroot'ed for each user at logon.

    For example, when user "bob" correctly types he's password at KDM, commands follows:

    1) pivot_root /home/bob /home/bob/.system

    2) export PATH=/.system/bin:/.system/usr/bin

    3) startkde

    People

    Victor Yacovlev originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
