    • CIM Enabled Shell

    Tags: InProgress, Management, SLES

    Description

    There already exists a binding between CIM and python in SUSE called pywbem. It would be nice to try to integrate this with python shell -- iPython.

    I'm currently working on a CIM to Python mapping called powerCIM. It provides a more intuitive mapping of CIM objects to Python than pywbem does (but it is still using pywbem as back-end). Integrating powerCIM with iPython is probably the way to go, because it has some nice features which are not in pywbem (e.g. if you have a SSHService class you can call SSHService.StopService() directly on the object).

    People

    Martin Mrazik originated this idea. He is currently experimenting with this idea.
