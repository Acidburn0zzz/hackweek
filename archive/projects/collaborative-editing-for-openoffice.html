    • Collaborative editing for OpenOffice

    Tags: Idea, Desktop, office

    Description

    Allow multiple people to work together on the same document in OpenOffice at the same time in the same visual space.

    The OO internals need to be examined because it does not use a "piece table" implementation underneath the word processor which would make this easier.

    People

    Michael Meeks originated this idea, Florian Reuter knows a lot about the relevant OO internals here. No one is currently working on implementing this idea (it's up for grabs!).
