    • Collaborative language learning

    Tags: InProgress, language

    Description

    Content: * vocabulary * drills * texts with additional language information ("daily czech") * grammar explanation * sound (and other multimedia?) content if I can convince the Prague colleagues to record it ;-) * all indexed and cross-referenced of course

    Frontends: * web interface * Qt client for offline use * palm client and conduit (later)

    Workflow: * have everyone create learning content * reviewer mode for native speakers of target language * comments / wiki-like notes for everything

    First target group are the czech learners in Nuremberg. The tool is meant to be language independent, but specific to language learning (generic learning tools are available already). Yes, I am aware that there are many vocabulary drill programs in existence, I'm using one of them ;-) They are limited though.

    Possible names so far are "OpenAutobus" ;-) and "Nerozumim", further suggestions are welcome.
