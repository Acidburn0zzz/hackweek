    • Config Tool for all PAM config files

    Tags: InProgress, Security, Management

    Description

    Currently we can only parse and modify /etc/pam.d/common-* files to modify the PAM configuration. But a lot of PAM modules are not for all, but only for special services (like pam_loginuid, pam_ck_connector, ...). So extend the current pam-config tool to allow to maintain such configurations. To get pam-config fully working and accepted, we also need to add support for all missing PAM modules.

    People

    Thorsten Kukuk originated this idea. He and Sven Schober are currently working on implementing this idea.
