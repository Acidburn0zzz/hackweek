    • Configure multi-button mice in Yast/Sax

    Tags: Idea, desktop, opensuse, yast, x11

    Description

    It would be nice if the SaX2 module that configured mice allowed for extra mouse buttons (buttons 6,7 etc as in MS Intellimouse explorer, Logitech MX series et al) to be mapped to arbitrary X commands/keys, e.g. document back/forward, shift, control and so on. This would provide a nice easy way to graphically configure these rather than diving into X configuration files, especially useful for new users.

    People

    vespas originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
