    • Content based indexing of files

    Tags: Idea, Desktop

    Description

    Content based indexing file. There will be a common folder to put all your data files. This tool will generate index based onits contents, will display those tags through interface. e.g. To store your music files, no need to make sub folders according to song mood, or singer etc. Simply put all ur music files in one folder. This tool will tag them with maximum possible way. So you can see your file under multiple folders at the same time. Your music will be catagorized by mood, by singer, by band etc at same time. User can see all these folders on interface at the same time. No need to organize your data file. :) Does it sound interesting....
