    • Creat a (cute?) SUSE-tan & put her in a desktop-theme

    Tags: InProgress, Desktop, pimpmydesktop, gnome, kde, HelpWanted

    Description

    The idea is to design a 'SUSE-tan'. So called 'OS-tan' are anime-style girls, each representing a different OS or different versions of one OS. Please see the link to wiki below for further information on OS-tans. Apparently there have been attemps to create a SUSE-tan before, but I don't like either of them.

    My aim is it to create a SUSE-tan and one or several desktop-themes centered around her.
