    • Create a calendar format which doesn't suck

    Tags: Idea, Office, Standard, Calendar

    Description

    Exchange of calendar data is still an adventure. The only format which comes close to an accepted standard is iCalendar. It still leaves a lot to desire, though.

    It would be great, if there would be a calendar format which doesn't suck. That's a real challenge, because calendar data includes monstrosities like time zones and recurrence rules. But maybe somebody has a smart idea how to solve this problem.

    I guess this project would be a good candidate for "heroic failure". ;-)
