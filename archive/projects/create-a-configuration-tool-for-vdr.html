    • Create a configuration tool for VDR

    Tags: Idea, Management, Server, Multimedia, HelpWanted

    Description

    VDR is a set-top-box like application for Digital TV (DVB). It is designed to work with full-featured cards or software mpeg decoders. The configuration is done through an on-screen-display menu. There is, however, also a very nice webfrontend called vdradmin-am and there are various plugins that allow network streaming of the mpeg stream. This works very well once configured correctly.

    The idea is to create a configuration tool for VDR that will enable users to configure VDR without using the OSD. It is yet to be determined if this should be done as an extension of vdradmin or in a separate tool.

    People

    Stefan Seyfried originated this idea. He will work on it during hackweek. Interested people, especially if they know anything about VDR or vdradmin are welcome :-).
