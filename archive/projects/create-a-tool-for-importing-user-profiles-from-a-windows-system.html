    • Create a tool for importing user profiles from a windows system

    Tags: InProgress, Installer, Windows

    Description

    It would be nice to have a tool which reads the available user profiles from a windows partition and creates the appropriate profiles for SuSE. Things like e-mails, contacts and private data should be imported to the new home directory.

    More things we could tease out of windows: * recently used applications (and add the linux equivalents to the kde or gnome menu) * installed application (and let the installers package selection depend on it)

    For gathering information from a windows partition we need to read the windows registry. Samba 4 contains tools for reading the windows registry.

    People

    Helmut Schaa originated this idea. He is currently working on implementing this idea.
