    • Universal Power Management Command Line Application

    Tags: Idea, Desktop

    Description

    Currently we have gnome-power-manager for the GNOME desktop and kpowersave for the KDE desktop. Idea is to extend the current available powersave (part of the powersave daemon) command line tool to be useable independently from any specific desktop environment. Additionally, it should become more like a universal power management command line tool with desktop session capabilities. This would include:

    - daemon mode - adjustment to plain C? - split from the powersave daemon package --> own project - make powersave daemon interaction an optional enhancement - features like desktop idle detection, battery monitoring - possibility to hook into any notification system like libnotify, knotify or libqnotify - a lot of small features like writing and reading to/from sysfs or procfs - tool should become "the universal power management application" for console users

    People

    Holger Macht originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
