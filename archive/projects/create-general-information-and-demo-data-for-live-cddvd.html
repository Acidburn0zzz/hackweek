    • Create General Information and Demo Data for Live CD/DVD

    Tags: Idea, Community, Desktop, openSUSE, Documentation

    Description

    Create general information and demo data which can be included on the openSUSE Live CD/DVD.

    General texts could include explanations of what eg Free Software, GNU, Linux, a distribution and openSUSE is.

    Demo data should be in different formats which people can open try with the installed applications (picture, music, videos, PDFs, office documents (text, spread sheet, presentation).

    People

    Stephan Binner originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
