    • Create something like andLinux but based on openSUSE instead of Ubuntu

    Tags: InProgress, UnixFeature, openSUSE, HW-Hacker

    Description

    //andLinux// is a distribution that runs //Ubuntu// seamless on Windows 2000 based systems (32-bit only). //andLinux// uses //coLinux// as its core, a port of the Linux kernel to windows (for details see http://andlinux.org). While I like the idea, I would rather prefer something using openSUSE. That's why I'll use the upcoming hack week to start creating something like it.
