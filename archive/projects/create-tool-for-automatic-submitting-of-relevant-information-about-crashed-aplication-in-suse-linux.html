    • Create tool for automatic submitting of relevant information about crashed aplication in SuSE Linux.

    Tags: Idea, openSUSE, QA, bugs

    Description

    Create a tool which would be integrated in SUSE Linux, for automatic submitting of reports with relevant information about crashed application in SUSE Linux. It could point to Bugzilla or some another information pool, where the QA or Package Maintainers will see that some application is crashing more often.

    This could be useful mostly during BETA testing.

    People

    Ladislav Michnovic originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
