    • Custom SysRq actions

    Tags: Idea, Kernel, HelpWanted

    Description

    A kernel patch or module would upon pressing alt-sysrq-x start a shell script somewhere from /etc or /bin (or at worst a binary if shell would be technically difficult). This could be used to recover frozen X without need for complete restart, perhaps also people who use network sysrq daemons could fine some use. So one could flexibly extend the range of possibilities in case of machine freeze.

    hajma originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
