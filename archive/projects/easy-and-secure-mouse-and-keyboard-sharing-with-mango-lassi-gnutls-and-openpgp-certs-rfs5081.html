    • Easy and secure mouse and keyboard sharing with mango lassi, GNUTLS and OpenPGP certs (rfs5081)

    Tags: Idea

    Description

    [[http://0pointer.de/blog/projects/mango-lassi.html|Mango lassi]] is a keyboard/mouse sharing daemon based that uses Avahi to discover other systems on the network. It is a great start but without a secure transport it is not practical. The secure transport will be GnuTLS using OpenPGP certs (rfc5081).

    A simple use case for mango lassi is a desktop machine controlling a single laptop on a single users desk. OpenPGP is used since it is more likely that OpenPGP is used on desktop machines rather than X509 certs. Using RFC5081 also simplifies the creation of secure keyboard/mouse shares between users who have each others key in the their trusted keychain.

    Possible collaboration with [[http://live.gnome.org/Seahorse/|Seahorse]]
