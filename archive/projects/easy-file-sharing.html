    • Giver - Easy File Sharing

    Tags: Done, TryMe, InBuildService, Desktop, Network, Gnome, FileSharing, JudgeMe, openSUSE, JudgeDesktop, JudgeBest, JudgeCommunity

    Lightning Talk

    Description

    Giver is a simple file sharing desktop application. People that have machines near you are automatically discovered and you can send files to them by simply dragging the files to their photo in Giver. There is no knowledge needed beyond what the person looks like if they are running Giver.

    Updated News

    Giver now integrates with the new HackWeek release of Tomboy. Notes can now be dragged and dropped to the same people and if they accept the note, a new note will popup in Tomboy... ROCKIN!

    Giver is now on the OpenSUSE Build service here

    If you want to just install it on your OpenSUSE 10.3 here is the repository.
