    • Enhance the desktop select. dialog in the installation by providing screenshots of those listed ones

    Tags: InProgress, installation, YaST, Graphics, Desktop

    Description

    It's very confusing to have the Desktop Selection Dialog without any pre-selected desktop. It's because we can't decide which we should pre-select, however, we could provide more information to the user just to make the decision easier.

    My idea is to change the dialog by using an Add-On product that would contain screenshots and a YCP-image-browser. This feature can be only supported with YaST in Qt or GTK+, ncurses do not support images ;-).

    People

    Lukas Ocilka originated this idea.

    Lukas Ocilka & Jakub Friedl are going to do the Add-On with screenshots but any help would be appreciated, e.g., from usability and look and feel side of view.
