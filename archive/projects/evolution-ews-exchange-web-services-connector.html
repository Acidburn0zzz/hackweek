    • Evolution EWS (Exchange Web Services) Connector

    Tags: Idea, Desktop, Evolution

    Description

    **EWS** opens new era of programming with Microsoft Exchange Servers. No more packet sniffing is required to write a client that can talk to Microsoft Exchange Servers. EWS comes with a well-written API documentation and follows the well known Web Services standards and allows clients to access the data store items through SOAP interface. Evolution has been a pioneer to provide connectivity to Exchange Servers and this idea is to keep it a pioneer in all aspects when it comes to connecting to Exchange Servers. EWS has a limitation that it will support only Exchange 2007 onwards.

    People

    Veerapuram Varadhan originated this idea and is currently working on implementing this idea (it's up for grabs!).
