    • Evolution Integration for ICECore Teaming and Conferencing

    Tags: Idea, InProgress, Desktop, Community, openSUSE, SLED, Evolution

    Description

    Evolution should be able to integrate with the ICECore teaming and collaboration server.

    Every user can be part of different teams. Once the user creates account with Evolution. Evolution should be able to create Tasks and Calendars of those teams in Evolution. Assume User A is part of Team1 and Team2. Then

    • *Calendar View of Evolution**

    Personal Calendar Calendar

    • *On This Computer**
    • *Team1**
    • *Team2**

    Personal Tasks Tasks

    • *Task View of Evolution**
    • *On This Computer**
    • *Team1**
    • *Team2**

    The calendar and the tasks view should fetch the respective items for the user's teams and hold it in EDS. Which means that Evolution should be able to show these data. Clock applet should show the users meetings and tasks for the selected day, etc.
