    • Evolution Mailing List Plugin

    Tags: Idea, Evolution

    Description

    Create a mailing list where all email address in a specific mail account are auto-added to a specific Contact List of your choice.

    Example usage: Blogsite has a form to allow visitors to subscribe to mailings. Mail is sent to a specific email account. Evolution then takes all new addresses in that account folder into a Contact List. You, the maintainer, then can send out periodic mailings without any membership maintenance needed whatsoever.

    Bryen Yunashko originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
