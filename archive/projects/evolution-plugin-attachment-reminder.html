    • Evolution plugin Attachment reminder

    Tags: JudgeMe, JudgeDesktop, Done, Desktop, Evolution, mail

    Description

    Add the functionality to Evolution to search for expressions like //"see attachment"//, //"see attached file"//, etc. and their localized variants (could be limited by language settings) at the time the users ignites the send command.

    If those expressions were used in new parts of the email (not in the >-marked parts) and NO file is attached to the email, a dialog should pop up to ask the user whether the email should really be sent this way, aka it shows a attachment reminder, similar to empty subject or empty body reminders.

    This should prevent the user from sending emails without attachment, although the main reason for the email was the attachment.

    User defined expressions would be a benefit.

    Could/should be implemented as a plugin.
