    • Extend TCC (Tiny C Compiler) beyond x86 and ARM

    Tags: Idea, compiler, powerpc, s390, ia64, ARM, amd64

    Description

    TCC (Tiny C Compiler) is a very fast and simple near-C99 compiler currently supporting x86 (to-memory and to-file (ELF and PE) compilation) and ARM (to-memory only). It is good enough to be able to compile the Linux kernel at boot time in 15 seconds and even has its own bootloader for that purpose. :)

    Extend it to support all the great architectures out there, and maybe implement to-file compilation on ARM, too. Judging from the amount of code necessary to support ARM, adding a new backend should be feasible in less than a week.

    People

    Ulrich Hecht originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
