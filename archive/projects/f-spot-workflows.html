    • F-Spot workflows

    Tags: Idea, Mono, Desktop, SLED, f-spot

    Description

    F-Spot today provides a set of photo management services that have been arranged in a particular mode that is suitable for photo browsing, tagging, searching and using as an editor for simple projects.

    It would be interested to provide alternative front-ends that would implement different workflows for the same database and the same rendering engine and editing options, some workflows that have been discussed in the past include:

    * Photo selection workflow: to aid photo developers in selecting best pictures, grouping together pictures, examining features on pictures. * Directory Photo browser: this already exists, but it is only available as power tool instead of being a different "facade" to F-Spot.

    People

    Miguel de Icaza originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
