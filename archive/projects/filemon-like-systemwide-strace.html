    • Filemon-like systemwide strace

    Tags: InProgress, Kernel, Desktop, UnixFeature, Performance, DevTool, JudgePerf, JudgeMe, TryMe, JudgeUnixAttack

    Description

    Filemon (http://www.microsoft.com/technet/sysinternals/FileAndDisk/Filemon.mspx) is a Windows utility that works like a systemwide strace(1). System administrators can use it like they would use strace, for example, to find why programs fail or crash if they aren't able to open a file.

    Systemtap lets us do global monitoring of system calls.

    Mortadelo is a quasi-clone of Filemon, based on a systemtap core and a GUI. We can later expand the program to do more interesting analysis, not just "show me all the system calls that happen".

    You can see screenshots starting with this blog entry: http://primates.ximian.com/~federico/news-2007-06.html#mortadelo
