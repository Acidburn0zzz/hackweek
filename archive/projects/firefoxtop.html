    • FirefoxTop: identify which tab is thrashing your cpu

    Tags: Idea, firefox, performance

    Description

    All too often, I notice that the load average on my system is shockingly high. Looking at top, I'll see that firefox is consuming 50% or more of my cpu, even though I'm off working in some other window. Given that I usually have 40-50 tabs open, trying to identify which one or more tabs is causing firefox to be busy is a significantly non-trivial task.

    It would be really useful to have a utility like top, slabtop, powertop, and xrestop for firefox to see which tabs are resource-sucking hogs.

    People

    Steve Beattie stole this idea from somewhere on the IntarWeb. No one is currently working on implementing this idea (it's up for grabs!).
