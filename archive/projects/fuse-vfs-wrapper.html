    • FUSE-VFS-wrapper

    Tags: Kernel, Filesystem, FUSE, osx, InProgress

    Description

    The Linux kernel includes quite a broad set of filesystem drivers, most of them unique to Linux. If you'd want to access a reiserfs or ext4 on BSD / Mac OS X / Windows you're more or less lost.

    An easy way to have all filesystems available on BSD / Mac OS X would be to provide an interface similar to the VFS layer in the kernel in userspace, which maps back to FUSE, which is supported in all major BSDs and OSX.

    As a first milestone I would suggest to get VFAT working. As soon as that works, ext2, ext3, and at some point the whole world.

    People

    Alexander Graf originated this idea and will start working on this next week. Stefan Assmann is gettin on it as well.
