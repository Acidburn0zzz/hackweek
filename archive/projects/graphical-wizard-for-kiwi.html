    • Graphical Wizard for KIWI

    Tags: Idea, DevTool, openSUSE, buildservice

    Description

    KIWI allows to easily create among others customized installation and live DVD/CDs of openSUSE. Create a graphical wizard which allows users to create their own flavor without certain/with certain packages (from the build service) without having to know about KIWI's XML configuration file.

    People

    Stephan Binner originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
