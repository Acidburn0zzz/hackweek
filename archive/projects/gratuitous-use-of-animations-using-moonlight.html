    • Gratuitous use of animations using Moonlight

    Tags: Idea, Desktop, pimpmydesktop

    Description

    With our Silverlight implementation (http://www.mono-project.com/Moonlight) we have a framework to do animations, video and audio that is relatively easy to program, that can be driven externally by an XML file and that will do smooth animations on a canvas.

    Although Moonlight was originally implemented to be used on a Web page to access Silverlight content, it is useful beyond the web, we can use it for our own desktop applications.

    We have a Gtk widget already written (this is what we use to test it).

    Some ideas would be:

    • Covert art browser for Banshee, animations, etc
    • Full screen image renderer
    • Full screen video player for Banshee, providing all the controls, and cutsey animations for it.
    • Find songs using a spacial representation of data.
    • Add effects to dialog boxes

    This is not practical per se, this is merely a mechanism to do user experience improvements with a relatively advanced language (and a relatively poor implementation at the time of this writing, with all of 2 weeks of work done so far :-)

    People

    Miguel de Icaza originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
