    • grub stage 1.5 for ext4 extents mode

    Tags: Idea, bootloader

    Description

    ext4 file system in extents mode does not compatible with ext2/3. Once root file system is formated as extents ext4, current grub 1.5 stage can not support it. Add a new grub stage 1.5 for ext4 extents mode for grub2 (and grub legacy if needed).

    People

    Coly Li originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
