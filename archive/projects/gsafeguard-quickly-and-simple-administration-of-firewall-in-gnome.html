    • GSafeguard - Quickly and simple administration of Firewall in GNOME

    Tags: InProgress, Security, Desktop

    Description

    If you wanna start/stop Firewall or configure it, you must run SuSEfirewall2 in terminal or select the firewall managerment from Yast, a little inconvenient.

    1. Applet for gnome that could start and stop firewall quickly.

    {{ ideas:GSafeguard_applet.png | GSafeguard Applet screenshot}}

    2. Provide a log list what blocked and allowed by firewall.

    {{ ideas:allow_packet.png | GSafeguard Allow packet menu screenshot}}

    3. A simple and friendly GUI for displaying and administering firewall rules. For example, user could right click one log in list, then from menu select allow the packet, after that some rules will be add to Firewall's rules.

    People

    Bin Li originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
