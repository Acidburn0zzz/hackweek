    • GUI for creating beamer style presentation

    Tags: Idea, Office, openSUSE, SLED, LaTeX, Desktop

    Description

    There is the excellent LaTeX Beamer Class which makes it pretty easy to create high-quality slides for presentations. Especially if you want to reuse content from existing LaTeX documents or you need formulas there is nothing better.

    To make editing Beamer presentations even easier it would be nice to have a GUI for it. This could contain of an editor which has some Beamer-specific functionality to handle slides, slide transitions, etc and a live preview of the result which is continuously rendered in the background.

    It would be fairly easy to implement something like that by using some of the available KDE components, like the editor and the PDF viewer KParts.

    People

    Cornelius Schumacher originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
