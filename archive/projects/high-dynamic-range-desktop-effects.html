    • High Dynamic Range desktop effects

    Tags: Idea, Desktop, PimpMyDesktop, X11, graphics

    Description

    Extremely pimpy, I know, but games these days use High Dynamic Range color processing to make things look *really* shiny or dark. We could include this feature in our desktop effects, and get another length between us and Vista's look and feel.

    Example: bumpmapped windeco with HDR highlighting for the active window.

    People

    Will Stephenson originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
