    • IM - Whiteboard

    Tags: Desktop, InProgress, HelpWanted

    Description

    Often when you communicate with others via IM, you need a way to visualize something. The idea is to implement a network flipchart, which gives each connected user the ability to visualize anything live to others. So you are able to hold network conference with the addition of the realtime flipchart, where you can save your results, demonstrate ideas or just to boost your arguments with visual stuff. The further step would be to think about implementing it into some IM protocols to use it within your IM application, or to leave it as standalone application.

    Some features: * live visualization (freehand, some shapes) * conflict handling (who has the pen...) * possibility to save it
