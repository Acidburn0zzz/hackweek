    • Image minimizer

    Tags: Idea, Virtualization

    Description

    Virtual images often are pretty big. It would be nice to have a tool to help making them smaller by removing unneeded packages and files. This could be done by analyzing at runtime which files are actually accessed. The result could be used as proposal what to remove and then fed back into Kiwi. Somebody at SUSE had written a tool to help with the analyzing of file accesses, but I forgot who it was.

    People

    Cornelius Schumacher originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
