    • implement metalink support in the download redirector

    Tags: Idea, openSUSE, Community, Server

    Description

    The [[http://en.opensuse.org/Build_Service/Redirector|download redirector]] (running at http://download.opensuse.org) could serve metalink files.

    Read this [[http://en.wikipedia.org/wiki/Metalink|wikipedia article]] [[http://www.metalinker.org|www.metalinker.org]] for more information.

    Generation of metafiles on the fly is not doable, because it involves hashing large files (iso images), but given the hashes of the files (or, better, hashes of individual chunks) are known, the rest of the metalink files could be generated on demand, as far as I can see. This would mean that the metalink files would be as up to date as the mirror database.

    The download redirector currently provides info like [[http://download.opensuse.org/distribution/10.2/iso/cd/openSUSE-10.2-GM-i386-CD1.iso?mirrorlist|this]]. It sorts the mirrors most suitable for a client based on country/region and a score. [[http://en.opensuse.org/Build_Service/Redirector|Here]] is more information about the redirector.
