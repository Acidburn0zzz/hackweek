    • Implement the DMTF Software Inventory and Software Update Profiles

    Tags: InProgress, Management, TryMe, InBuildService

    Description

    Implement the Software Inventory and Software Update Profiles for SUSE Linux. Use the PyWBEM Python provider interface, and rpm-python.

    Packages are in the [[http://software.opensuse.org/download/systemsmanagement:/wbem/|Build Service]]

    This [[http://pywbem.wiki.sourceforge.net/Provider+QuickStart|QuickStart Guide]] may help to get things installed.

    People

    Bart Whiteley originated this idea. Bart Whiteley is working on this idea.
