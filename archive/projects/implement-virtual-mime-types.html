    • implement virtual MIME types

    Tags: Idea, Graphics, IdeaPool, openSUSE, SLED, HelpWanted

    Description

    Many applications use plugin system. It allows to extend supported MIME type list to many other applications by adding one additional packages.

    Examples: Adding raw-plugin adds support for image/x-dcraw to GIMP.

    Adding new gdk-pixbuf-loader module adds support for new file format to all viewers using gdk-pixbuf loader.

    It's proposed to implement virtual mime type in frame of shared-mime-info, which will support this definition.

    People

    Stanislav Brabec originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
