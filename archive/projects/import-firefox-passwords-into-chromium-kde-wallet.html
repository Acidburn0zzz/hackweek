    • import firefox passwords into chromium kde wallet

     

    Description

    somehow import firefox passwords into chromium.

    They can be exported using the Password Exporter plugin.

    They in theory can be imported into KDEWallet using the KDE Wallet password integration plugin

    In practice, the imported firefox passwords are in another wallet folder than the chromium passwords.
