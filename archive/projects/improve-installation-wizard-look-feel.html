    • Improve installation wizard look & feel

    Tags: Done, Desktop, Graphics, Installation, Design, yast

    Description

    SUSE installation wizard has kept the same look&feel for many years. With Qt4, there are new possibilities of deploying graphics, design and effects in order to give users the best possible first impression :-) since installer is the first thing they usually see in SUSE. The idea is to come up with new look & feel of installation wizard and to implement some basic mockups using Qt4

    People

    Katarina Machalkova originated this idea. This has been done by Stephan Kulow for openSUSE 11.0 Alphas already.
