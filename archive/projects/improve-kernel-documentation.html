    • Improve Linux Kernel documentation

    Tags: Idea, Community, Kernel, Documentation

    Description

    Some portions of the Linux kernel are hard to understand mainly because of little (or no) documentation or comments.

    Improving Kernel documentation can be done in any of the following ways:

    • Documentation patches to any specific kernel subsystem
    • Additions/Modifications to /usr/src/linux/Documentation/ files in Linux Kernel
    • Translating /usr/src/linux/Documentation/ files (Though inclusion of translated documents with the kernel source code doesn't make much sense to me, I think atleast translating document and making it available in some form helps)
