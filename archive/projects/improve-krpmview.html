    • Improve KRPMView

    Tags: Done, devtool, desktop, KDE, RPM

    Description

    Clicking on an rpm file in Konqueror starts KRPMView with the tabs "Description", "Changelog" and "Files".

    The suggestion is to add "Technical Data" and "Dependencies" tabs like known from YaST package manager.

    Extra challenge: Add a tab "File Size View" which visualizes the file sizes in the rpm with the fsview kpart.

    People

    Stephan Binner originated this idea and worked on it. New kdebase3-SUSE package is submitted to Factory.
