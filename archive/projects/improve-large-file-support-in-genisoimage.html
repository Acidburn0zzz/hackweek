    • Improve large file support in genisoimage

    Tags: Idea

    Description

    Recent genisoimage has this support for files bigger than 2 GB:

    -allow-limited-size

    When processing files larger than 2GiB which cannot be easily represented in ISO9660, add them with a shrunk visible file size to ISO9660 and with the correct visible file size to the UDF system. The result is an inconsistent filesystem and users need to make sure that they really use UDF rather than ISO9660 driver to read a such disk. Implies enabling -udf.

    This can be further improved:

    • add option to write only the UDF filesystem
    • implement multi-extent files (ISO9660 level 3)

    People

    Vladimir Nadvornik originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
