    • improve and clean Palm Tungsten|T3 kernel support

    Tags: Kernel, Mobile, LowPower, InProgress, HW-Hacker

    Description

    I want to improve [[http://en.wikipedia.org/wiki/Tungsten_Handheld#Tungsten_T3|Palm Tungsten|T3]] in [[http://www.hackndev.com|Hack&Dev]] kernel. As this [[http://handhelds.org/moin/moin.cgi/PalmTungstenT3Status|site]] can show, almost all has been written and it's working somehow. Now it's time for second step - make drivers clean and stable. Most drivers are buggy and work with T3 is not as comfortable as it would. I'd like to * make bluetooth working * finish reworked button driver * make SD driver independent on CPU speed (some cards needs to have set lower speeds) * make IR port usable
