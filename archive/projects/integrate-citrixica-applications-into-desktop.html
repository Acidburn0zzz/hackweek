    • Integrate Citrix/ICA applications into desktop

    Tags: Idea, Desktop, openSUSE, SLED, Network, HelpWanted

    Description

    Currently, I think people usually launch Citrix applications via the web-based application browser. I think it would be great if we could integrate a citrix app browser into our desktop application browser. So maybe you would have a combo box of different app servers in the sidebar, and selecting one would show the normal stuff in the browser pane. Of course, double clicking the app would launch it just like a normal one.

    People

    James Willcox originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
