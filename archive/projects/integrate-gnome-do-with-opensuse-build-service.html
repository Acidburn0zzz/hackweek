    • Integrate GNOME Do with openSUSE Build Service

    Tags: Idea, openSUSE, Community, DevTool, BuildService

    Description

    Currently, managing Build Service projects is done using console application, web client or an GNOME/KDE app, which is not fully functional yet, like Eclipse plugin.

    Some basic functions can be done using GNOME Do, but we need plugin fo that.

    Triggering rebuild, downloading package, notification about build success/failure (since Do is working in background, can send notification via libnotify).

    People

    Jakub 'Livio' Rusinek originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
