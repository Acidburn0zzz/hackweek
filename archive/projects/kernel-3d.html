    • Kernel 3D

    Tags: Kernel, Graphics, Idea, devtool, 3d

    Description

    The idea is to represent todays kernel in a 3 dimensional way. I think of cube within a small 'space ship' with that you can jump/fly around and have a overview. All kernel-rings and their corresponding files should be displayed. Their relationship should also be viewable. The aim is to see cubically how the dependencies are. Furthermore it should be able to fly around and to go deep at least at the function level.

    In a next step it would be great, if you can display the current used subsystems of the kernel.

    People

    Patrick Kirsch originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
