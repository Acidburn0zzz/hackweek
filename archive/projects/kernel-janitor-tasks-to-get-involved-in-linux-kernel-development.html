    Description

    The Linux Kernel Janitor project is a great way to get involved in the Linux kernel development community. It has a large list of simple "cleanup" type changes that the kernel developers want to see done to the code tree. All of these tasks can be picked up in a short amount of time and enable you to get your name in the Linux kernel change logs and you will be on your way to being a Linux kernel developer.

    The list of tasks and instructions on how to get involved can be found at http://janitor.kernelnewbies.org/
