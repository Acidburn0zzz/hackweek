    • libvirt support installation from Amazon machine images

     

    Description

    Amazon Machine Images (AMIs) are preconfigured with an ever-growing list of operating systems. It provides hundreds of free and paid images for user to use and create virtual machine. If libvirt can handle AMI imags, then with virt-install, we can allow users to create most kinds of guest OSes (OpenSolaris, RedHat, Fedora, Debian,...) based on those AMI images.
