    • Live-DVD: Fast desktop switcher

    Tags: Idea, openSUSE, usability

    Description

    If we happen to create a Live-DVD with installation features (http://idea.opensuse.org/content/ideas/live-cddvd-install) why not include a nice way to choose and switch between KDE and GNOME?

    IMHO this is much better than swaping Live-CD for each environment. It would take two things:

    • A modified Login Screen; a mockup is in the wiki[1].
    • A simple way to switch between the environments (including maybe a cube effect for the eye candy addicts).

    This would not just be a catch-up to existing projects, this would be leapfrogging!

    People

    Joerg Kress originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
