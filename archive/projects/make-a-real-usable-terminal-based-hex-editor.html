    • Make a real usable terminal-based hex-editor

    Tags: InProgress, DevTool, HelpWanted, HW-Hacker, reversing

    Description

    Make a usable ncurses or slang-based editor which works with arbitrarily large files and compiles cleanly on 64-bit platforms. Integrated disassembler for multiple platforms would be great too.

    The basic idea is that there are quite a few editors out there, but all of them are unusable in a way. We might also pick an existing hex-editor and fix it so that it fulfils at least the most basic requirements.

    People

    Petr Tesarik originated this idea and is working on it.
