    • Make all keyboards and buttons working in X

    Tags: InProgress, Kernel, Desktop, Mobile, X11, openSUSE, SLED, SLEPOS, multimedia, keyboard, evdev

    Description

    With introduction of Linux kernel 2.6, there are three different modes of keyboard operation: raw mode (sends raw AT keycodes, compatible with kernel 2.4, does not work non-AT keyboards), soft raw mode (emulates AT keyboard, special keys don't work in this mode properly in X yet), evdev mode (new Linux kernel driver, X implementation does not work properly).

    This project tries to create unique key mapping and key tables for most common keyboards on both kernel level and X level.

    People

    Stanislav Brabec originated this idea, Vojtěch Pavlík is an author of kernel evdev driver. Stanislav Brabec and Hans de Goede are currently working on implementing this idea.
