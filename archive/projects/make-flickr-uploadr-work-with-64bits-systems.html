    • Make flickr-uploadr work with 64bits systems

    Tags: Done, Desktop, Web20, openSUSE

    Description

    There's a Linux version of the Flickr Uploadr available in the buildservice here https://build.opensuse.org/package/show?package=flickr-uploadr&project=home%3Amcadet%3Agnome-tools. However it currently doesn't work with 64bits systems. It seems that there are some parts of the code that assume a 32bits architecture. It would be nice to fix this issue to bring more Flickr love to openSUSE users :)

    People

    Mathieu Cadet originated this idea. No one is currently working on implementing this idea (it's up for grabs!).

    I need a hacker who knows about C++ on 32&64bit platforms. The involved C++ codebase is quite small so I guess it would not take long for an experienced person to find and fix the issue.
