    • make libexif more usable

    Tags: Idea, Desktop, Graphics, openSUSE, SLED, HelpWanted

    Description

    Current version of libexif is nat well maintained and has really bad support for MakerNote and other special metadata. On the opposite there is exiftool (and exifprobe) which knows more but does not provide library to be integrated.

    Also tag writing should be improved to not break internal structure of some manufacturer-specific tags.

    People

    Stanislav Brabec originated this idea. Mauro Parra is interested in working in this idea. No one is currently working on implementing this idea (it's up for grabs!).
