    • Make libzypp support downloading all packages to a local disk cache before starting to install.

    Tags: InProgress, Libzypp

    Description

    It's a frequently requested feature to support downloading packages to a local disk cache before starting to install.

    While this is fine when maintaining a system, it might not be appropriate on bare metal installation or system upgrade. So it has to be made configurable by the application.

    People

    Michael Andres is currently working on implementing this idea.
