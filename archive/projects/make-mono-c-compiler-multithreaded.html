    • Make Mono C# compiler multi-threaded application

    Tags: Idea, devtool, mono, compiler

    Description

    Both Mono C# compilers are single threaded applications. It would be nice to get some benefits from more and more often used multi-cores CPUs and modify the compilers to utilize all possible machine power.

    The easiest way is probably start with the parser which could fully run in separate thread.

    People

    Marek Safar originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
