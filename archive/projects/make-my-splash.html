    • Make my splash

    Tags: InProgress, TryMe

    Description

    A application where you can use any picture to have it as grub/boot/KDE splash. Best would be to just drag and drop it and the user can decide if he wants it as grub/boot/KDE splash and as background.

    People

    Marco Michna originated this idea and is working on the project.
