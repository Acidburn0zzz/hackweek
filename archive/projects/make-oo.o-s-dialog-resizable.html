    • Make OO.o's dialog resizable

    Tags: Desktop, office, InProgress

    Description

    Currently, none of OO.o's dialogs (except for the native file picker) are resizable *by design*. We really need to make OO.o's dialogs resizable. While we're at it, we should also make an intuitive UI dialog designer, so that we don't have to write code to create and modify a dialog. We also need to develop a converter script to migrate the existing 700 dialog resource files into new XML format for the new layout engine.

    People

    Michael Meeks originated this idea, and is actively developing this. This is one of GSoC tasks, and a student has already been assigned to work on this task. Kohei Yoshida will take part in this task during the hack week.
