    • Make openSUSE Build Service Accessible by YOU

    Tags: Idea, Community, openSUSE, DevTool

    Description

    Ubuntu and Debian make it trivial to get nearly any package that someone has built for those distros.

    SUSE has the lovely build service, and bunches of repositories that one can point at. However the http://packages.opensuse-community.org/ site does not interop with the YaST software management tools. The web site there indexes and lets you search various repositories, but you cannot add a single repository that will let YaST just search them all.

    Idea: add a meta-repository so that the user can just type a progrma name into YaST and download the package if it is anywhere in the build service.

    People

    Crispin Cowan originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
