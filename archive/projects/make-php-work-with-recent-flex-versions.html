    • make php work with recent flex versions

    Tags: Idea, LAMP

    Description

    The php language scanner requires a certain (old) flex version, because it relies on internals of that version to implement a reentrant scanner. This causes problems for us (we have to keep that old version just because of php) as well as for users who want to rebuild php (they have to make sure they have the right version installed). So this idea is about porting the php scanner to recent flex. Given that flex has now native support for reentrant scanners, it shouldn't be too difficult.

    Note: I'm talking about the lex-style scanner generator, not the Adobe product :-)

    People

    Michal Marek is going to work on this during the hackweek. The idea itself isn't anything new actually.
