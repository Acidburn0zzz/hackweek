    • Make the desktop metadata aware

    Tags: Idea, Desktop, Metadata

    Description

    There are a lot of programs use metadata to organize document. F-Spot for example, or Rhythmbox. But each and everyone of those programs creates its own metadatabase.

    Why not create a unified desktop-agnostic framework for metadata? And make the desktop applications use that framework.

    People

    Raphael Bosshard originated this idea. No one is currently working on implementing this idea (it's up for grabs!).

    Related Materials

    http://www.gnome.org/projects/tracker/ has already started working in that direction.

    [[ http://nepomuk-kde.semanticdesktop.org/xwiki/bin/view/Main/ | NEPOMUK ]] is an existing desktop-agnostic metadata store.
