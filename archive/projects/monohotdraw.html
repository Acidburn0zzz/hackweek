    • MonoHotDraw: structured graphics library

    Tags: Graphics, DevTool, Mono, Idea, InProgress, TryMe

    Description

    MonoHotDraw is a two-dimensional graphics framework for structured drawing editors. It can be used to create graphic editors for UML, Mind Mapping, Flowcharts, Animation and Vector Graphics.

    People

    - Manuel Ceron was working on it - Mario Carrion is working on it

    Manuel Ceron and Mario Carrion originated this idea.
