    • Move YaST tutorials from forgeftp.novell.com to openSUSE WIKI

    Tags: Idea, Community, openSUSE, DevTool, YaST, TryMe, Done

    Description

    There are currently two tutorials at http://forgeftp.novell.com/yast. They are stored in our external YaST SVN and built for every release. That's not needed because those tutorials don't change. Moreover, WIKI is closer to the community and developers expect it there.

    Totorials...

    • are currently here: http://svn.opensuse.org/svn/yast/trunk/doc/tutorials/
    • can be seen here: http://forgeftp.novell.com/yast/doc/SL11.0/tutorials/
    • have better be placed here: http://en.opensuse.org/YaST/Development#Tutorials

    People

    Lukas Ocilka originated this idea. No one is currently working on implementing this idea (it's up for grabs!).

    Marek Stopka is working on this idea.
