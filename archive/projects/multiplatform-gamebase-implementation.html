    • Multiplatform GameBase implementation

    Tags: Idea, games, emulator, Desktop

    Description

    {{http://www.bu22.com/img/gb_s.jpg}}

    GameBase is an excellent emulator frontend for which an inordinate number of excellent game databases are available, most notably GameBase64. It is open-source, but written in Visual Basic 6, limiting it to Windows. There is a Java implementation (also open-source), but it does not use the original (MS Access) databases, making it a dead end: Windows users won't be able to use the databases you created. (Besides, the conversion from MDB didn't work for me.)

    Create a cross-platform implementation of GameBase that works on the original database format, ensuring interoperability with the original Windows program. There is an (open-source) library handling the database files used, so the task is essentially limited to recreating (and/or enhancing) the user interface.

    People

    Ulrich Hecht originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
