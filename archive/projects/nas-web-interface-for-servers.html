    • NAS web interface for servers

    Tags: Idea, Server, SLES, storage

    Description

    I would like to have a lightweight web interface to configure a NAS server - similar to the FreeNAS (see http://www.freenas.org ) Screenshots under http://www.freenas.org/index.php?option=com_content&task=view&id=15&Itemid=28

    While FreeNAS is based on FreeBSD, I would love to have such an interface for the SLES.

    We should be able to configure:

    • SAMBA
    • NFS
    • FTP
    • AFP (Apple)

    at a glance with interfaces to user management, permissions, etc.

    People

    Mihnea Istinie originated this idea. No one is currently working on implementing this idea (it's up for grabs!).

    The FreeNAS interface is PHP, and that's the point where I would start. In the end I can imagine an appliance/VM on top of SLES.
