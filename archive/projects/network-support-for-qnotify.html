    • network support for qnotify

    Tags: Network, Qt, Desktop, InProgress

    Description

    qnotify is a lightweight desktop notification application using only the Qt toolkit. The goal of this project is to give it network support so that it is possible to trigger notifications from remote machines.

    People

    Andreas Bauer originated this idea and is currently working on it.
