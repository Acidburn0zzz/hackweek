    • network topology drawer

    Tags: InProgress, Network, Graphic

    Description

    This project is about the implementation of a tool for easy and automated creation of network topology graphics. There is already a tool for similar things, but it does not produce pretty pictures for network graphics. http://www.graphviz.org

    The idea is to have a small text file where you can add groups or single systems and the relations of them.

    Example:

    group{ name="workstations" members="workstation workstation workstation workstation" } group{ name="router1" members="router" } group{ name="router2" members="router" } group{ name="webserver" members="webserver"}

    //relations: router1 => workstations; router1 => router2; router2 => werbserver;

    People

    Christian Schneemann originated this idea. He is currently working on implementing this idea.
