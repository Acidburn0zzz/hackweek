    • New desktop sounds, including IM

    Tags: Idea, Sound, Audio, Desktop

    Description

    We need nice, pleasant sounds for desktop events. The sounds should be high quality and descriptive for the event taking place. Also, only special events should have sounds (clicks and button presses should not, for instance...)

    People

    Garrett LeSage originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
