    • NNTP Back End for Web Forum

    Tags: Idea, Web, PHP, HelpWanted, NNTP

    Description

    Most web forums base on a database back end like MySQL. Having an NNTP back end would enable access to articles as well via NNTP clients. The idea is to exchange the database back end of a web forum by a NNTP backend. Current idea is to use phorum as web forum since there are several database back ends which (hopefully) enables an easy addition of a NNTP back end.

    People

    Holger Sickenberg originated this idea. He is currently working on implementing this idea.
