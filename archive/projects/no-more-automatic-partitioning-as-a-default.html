    • No more automatic partitioning as a default

    Tags: Idea, installation, partitioning, opensuse

    Description

    This is not a good idea to enable automatic formating/deleting/creating partitions using existing disks as a default. Most of the people install Linux/openSUSE under existing partitions and mostly they do not know how to change easly this, sometimes they just ignore this option and loose all data on the disk.

    opneSUSE instalator should just ask the user what to do:

    1. Install openSUSE on a existing partition (DEFAULT) 2. Automatic Proposal 3. Advanced options

    1# without '/home' and 'swap partition' - just '/' and swap file! 2# the same as it is right now 3# Partitioning tool

    People

    Sebastian Furdal originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
