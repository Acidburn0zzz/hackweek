    • Open source Malone

    Tags: Idea, DevTool, openSUSE, bugs

    Description

    Canonical has built a multi-project bug tracking system called Malone that makes it easy for people to track bugs in multiple bug trackers at the same time, and assists distros and others with tracking each other's work, and getting things upstream.

    Unfortunately, it's proprietary.

    Someone could build an open source replacement.

    People

    Nat Friedman originated this idea.
