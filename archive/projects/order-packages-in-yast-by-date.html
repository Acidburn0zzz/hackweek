    • Order packages in Yast by date

    Tags: Idea

    Description

    Ordering packages by date in Software management would make maintenance lot less tedious task. Currently installation dates can be seen in one of the tabs below package selection box, but it would be lot easier if package dates could be seen on the selection box view. This would ease uninstalling of packages a LOT, when you could see straight away e.g. latest installed packages, or packages installed on a certain date for that matter.

    People

    Lassi Väätämöinen originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
