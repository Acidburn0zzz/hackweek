    • Port the neutrino STB GUI to dreambox machines

    Tags: Done, Multimedia, HelpWanted, HW-Tester

    Description

    The Digital TV (DVB) set top boxes from dream multimedia are linux based. They are shipped with a GUI called Enigma that is not too nice to use (IMHO). There is another GUI for Linux based STBs called Neutrino which is very popular on the dbox2. Since the dreambox uses an older API for their DVB drivers, the GUIs cannot be exchanged too easily: dbox2/Neutrino has already moved to the new DVB API while dreambox is stuck at the old one. Finish the port of Neutrino to the old API and submit as much as possible for upstream inclusion in the project's CVS repository.

    People

    Stefan Seyfried originated this idea. He implemented it during hackweek. Most of the stuff should be working fine now with the code from CVS, the additional patches still need to be posted somewhere.

    Testers with different dreambox models than the DM500S available for testing are very welcome.
