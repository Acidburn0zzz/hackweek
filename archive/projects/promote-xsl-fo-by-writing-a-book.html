    • Promote XSL-FO by Writing a Book

    Tags: InProgress, HW-Docs, HW-Designer, Documentation, XSL-FO, XML, DocBook

    Description

    XSL-FO is an intermediate format that lies in between from XML on one end and PDF on the other end. The format goes beyond the capabilities of CSS but is sometimes hard to use. Usually it is used to format DocBook and the like to PDF.

    Currently there are no (free) books about XSL-FO that tells the readers how to create page layouts, lists, tables, etc. in a brief, solution oriented way.

    People

    Thomas Schraitle originated this idea and and will be working on this.
