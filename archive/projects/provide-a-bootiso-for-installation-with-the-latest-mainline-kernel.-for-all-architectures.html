    • provide a bootiso for installation with the latest mainline kernel. for all architectures.

    Description

    * build and package the latest mainline snapshot on a bootable iso image

    * use the currently availabe .config files from kernel CVS

    * decouple the build from openSuSE release cycles

    * provide logs with build failures to interested developers
