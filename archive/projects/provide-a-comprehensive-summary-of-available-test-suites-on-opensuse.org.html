    • provide a comprehensive summary of available test suites on opensuse.org

    Tags: Idea, Community, QA, Desktop, Performance, DevTool, openSUSE, HelpWanted

    Description

    There are a lot of test tool matrixes out on the internet, some are outdated, some not applicable, some are moving quite fast.

    My vision is that we provide the community some entry point to help with testing openSUSE. Together with some yet to be published documentation around testing (provided by OPS-QA, this is wip) we might really get some more spin here.

    The main task of this idea is to find out a feasible way to maintain the content - is a wiki sufficient for the needs or do we need some other backend.

    People

    Olli Ries originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
