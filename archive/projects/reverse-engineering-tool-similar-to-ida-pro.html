    • Reverse Engineering tool similar to IDA Pro

    Tags: InProgress, reversing, devtool, bugs, security

    Description

    Write an open-source reverse engineering tool with a feature set close to that of IDA (Interactive Disassembler) Pro. There has been an attempt to create such a thing, called "lida" (Linux Interactive Disassembler), but the project is stuck in pre-alpha state, and one can think of any number of reasons for that, such as Perl, Tk, libdisasm and pretty much every feature of the actual implementation. The concept is sound, though, and the homepage is worth a look. If you try to make it better, use a real programming language, a real GUI toolkit, and a real disassembly backend supporting more than one architecture.

    People

    Ulrich Hecht originated this idea. Petr Baudis (pasky) snatched it. Petr Tesarik (ptesarik) is also interested and wants to contribute.
