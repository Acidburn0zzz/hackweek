    • 'Safe' Sleep

    Tags: Idea, Mobile, openSUSE, SLED, Kernel

    Description

    We currently expose the two types of suspending a laptop computer -- sleep and hibernate. The user shouldn't need to make this choice.

    Suspend has obviously advantages in speed, but it's tricky in that you may lose data if you run out of battery. If the snapshotting onto disk could be done fast enough, we could always do it, but use suspend to RAM unless there is little battery left. So using suspend to RAM, but have a safe 'backup'.
