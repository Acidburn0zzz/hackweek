    • Scanner Buttons

    Tags: Idea, Graphics, Desktop, Drago, hardware

    Description

    Scanners often come with some "shortcut buttons" where people are used to simply press and an action like 'send scanned image via email' is started.

    These buttons should work and should be configured on a 'at-home' system although they can arise privacy/security problems somehow. There is already an upstream solution, link see below.

    People

    Klaas Freitag originated this idea.
