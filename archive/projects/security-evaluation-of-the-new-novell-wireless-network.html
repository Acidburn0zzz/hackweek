    • Security evaluation of the new Novell Wireless network

    Tags: Idea, Security, QA, Mobile, Network, openSUSE, SLED, Done

    Description

    The Novell wireless network uses an advanced setup with WPA2-Enterprise, using EAP-PEAP and MSCHAPv2 for authentication through a RADIUS server. This design isn't easy to implement right, and so we will check whether it is really as secure as it purports to be. And it has to be - it is a direct door to Novell's internal network.

    People

    Vojtech Pavlik and Jiri Kosina have worked on this idea, and have finshed the evaluation.
