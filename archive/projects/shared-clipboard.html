    • Shared Clipboard

    Tags: Idea, Desktop, HW-Hacker, Network, Mobile

    Description

    It is increasingly common to use more than one desktop/computer at the same time. And sometimes we want to copy/paste between two computers. For example, one may use a computer for hacking and another may be running the irc client. To ask about the error message in irc, it is difficult to copy paste from the other desktop.

    Extend klipper, glipper or add a new component that would allow configuring shared clipboards over network without any central server or Internet access. For example ctrl-c/ctrl-v can copy/paste to the local clipboard while shift-c/shift-v can copy/paste using the shared clipboard.

    People

    Nikanth Karthikesan originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
