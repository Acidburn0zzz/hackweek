    • Shared lightweight, public, web-based event calendar

    Tags: Web20, InProgress

    Description

    People from different organizations should be able to maintain a list of events. There should be at least two kinds of views: public events and internal events. These list should be publicly viewable for everybody. Every organization can integrate the event list to its own webpage. Im-/export of iCalendar files, RSS feeds etc. is, of course, mandatory.

    People

    Christopher Hofmann originated this idea and is currently working on implementing this idea.
