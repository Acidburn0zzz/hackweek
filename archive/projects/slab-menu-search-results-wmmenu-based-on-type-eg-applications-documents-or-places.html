    • Slab Menu Search Results Within the Menu Based on Type (eg Applications, Documents or places)

    Tags: Desktop

    Description

    Instead of having to click on the "More Applications", "More Documents", "More Places" buttons on the SLAB menu, why not integrate the search text box so that you can type in for example the application name and the recent and favorites list is replaced by a search results list within the menu which is specific to the tab you are in (Applications, Documents, Places). This would eliminate the need to click on the "More" category button so often. Also when adding favorites application It would be good if you had the option of adding it to the normal favorites area or the "System" menu area.

    People

    Gareth Jordan originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
