    • Speed up OGG/Vorbis encoding

    Tags: Desktop, sound, multimedia, InProgress

    Description

    One of disadvntages of OGG/Vorbis codec is its encoding/decoding speed. Especially, the encoding is significantly slower than MP3. There are several attempts to improve the encoding speed, for example, using SIMD ops. Unfortunately, they are not available on Linux. It'll be nice if we have a faster encoder.

    People

    Takashi Iwai originated this idea. Takashi Iwai is currently working on implementing this idea.
