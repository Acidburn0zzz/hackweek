    • save ufraw settings in xmp metadata

    Tags: Idea, InProgress, desktop, camera, Photography

    Description

    ufraw is a tool for "developing" raw files from camera. It should be possible to store the setting used for the processing together with the photo and reproduce or adjust the process later.

    xmp seems to be ideal format for it.

    UPDATE 19.2.2008:

    I have partially working patch. I decided to use xmp support in libexiv2, because this library is already used in ufraw. The xmp support in libexiv2 0.16 has some bugs, so the patch now works only with latest svn version.
