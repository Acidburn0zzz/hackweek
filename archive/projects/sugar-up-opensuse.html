    • Sugar up openSUSE

    Tags: Done, InBuildService, TryMe, Desktop, openSUSE, LowPower, HelpWanted, HW-Tester, HW-Packaging

    Description

    Build OLPC's Sugar for openSUSE

    First working snapshot: {{ideas:xo.png|}}

    People

    Jigish Gohil(cyberorg) originated this idea and is currently working on implementing this idea along with C Rodriguez(judas_iscariote).

    Get it

    zypper sa http://download.opensuse.org/repositories/devel:/languages:/python/openSUSE_11.0/ python

    zypper sa -k http://download.opensuse.org/repositories/home:/cyberorg:/sugar/openSUSE_11.0/ sugar

    zypper refresh

    zypper in sugar

    You should now have sugar-shell available at the login screen.
