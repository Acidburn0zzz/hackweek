    • Sync Evolution and Tomboy with iPod

    Tags: Idea

    Description

    My iPod has support for Notes, Address book and Calendar (iPod Nano). It would be nice if I could sync Tomboy with Notes on iPod, address book + calendar from Evolution to iPod.

    Evolution also has notes, which could be synced. But as a Tomboy user I want Tomboy ;)

    Saving Notes on iPod is as easy as saving a ascii-text-file in /Notes/.

    Saving Calendar entries is as easy as saving vCal files in /Calendars/.

    Saving contacts, save vCard files in /Contacts/.
