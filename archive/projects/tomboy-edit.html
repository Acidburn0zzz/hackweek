    • Tomboy Console Editor

    Tags: Idea, TryMe, Done

    Description

    A command-line editor for Tomboy notes. Similar to ''crontab -e'' or ''visudo'' this program pulls the specified note into a temporary file, opens your editor (whatever is in the ''EDITOR'' variable) and sends the result back to a running tomboy instance. Tomboy has a DBus interface for doing this kind of thing.

    Some users seem to like that they can edit their notes on the disk. This would become impossible if Tomboy moved to some kind of database back-end.

    This doesn't work if tomboy isn't running. Something may have to be done about that at some point. Still it's a fun idea and easy enough to hack up in a day.
