    • Totem should be able to fetch subtitles automatically

    Tags: Idea, Desktop, GNOME, Totem, Subtitles

    Description

    The idea is how the title reads - Totem should be able to automatically fetch subtitles to the currently played video - just like Banshee can fetch lyrics to music! In chosen language, with additional options (like for hearing impaired, so on). This would be nice, but it would also be good for accessibility reasons.

    People

    Ilya Ilembitov originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
