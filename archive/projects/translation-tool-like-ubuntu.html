    • Translation tool like Ubuntu's Rosetta

    Tags: Idea, Community, i18n

    Description

    Rosetta allows community users to submit translations for their language and for translators to approve/reject the submissions. With coolo's language bundle work I know this would be easier since we can inject extra translations into those bundles.

    Coolo believes though that the quality is not good for these translations and it upsets upstream projects because it siphons translation resources downstream.

    People

    JP Rosevear originated this idea, Stefan Kulow has some strong opinions on it. No one is currently working on implementing this idea (it's up for grabs!).
