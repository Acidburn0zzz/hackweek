    • VESA DDC/CI - KDE Applet

    Tags: Idea, Desktop, openSUSE, SLED, kde

    Description

    Monitors can be controlled by software using the VESA DDC/CI protocol. There's already a command line interface for that, called ddccontrol (http://ddccontrol.sourceforge.net/).

    Quote from the homepage: ''DDCcontrol is a program running on Linux, used to control monitor parameters, like brightness and contrast, by software, i.e. without using the OSD (On Screen Display) and the buttons in front of the monitor.''

    The idea is to create a KDE frontend for the ddccontrol library, possibly a taskbar applet. It would be quite cool to change the brightness, contrast and color temperature of your screen with a single click in the KDE kicker :)

    Features it should have:

    • Presets
    • Global keyboard shortcuts
    • Eventually, the application should have a DCOP or dbus interface, at least for loading presets.

    Additional ideas regarding DDC/CI:

    • ncurses alsamixer-like DDC/CI interface
    • Bluetooth/LIRC integration

    People

    Wolfgang Frisch originated this idea. No one is currently working on implementing this idea (it's up for grabs!). The project would be best done by someone who's well versed with the KDE libraries.
