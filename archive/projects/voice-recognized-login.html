    • Voice recognized login

    Tags: Idea, Desktop, sound, security

    Description

    The user should be able speak a "KEY" and the GDM should recognize the voice and validate the key and login the user. This should apply for the Lock Screen as well.

    • Voice validation: Even if my friend knows the key, unless he speaks my tone, he can't login.
    • Key validation: Key can be "Sivaji The Boss" which needs to be matched against what the user spelled. So that some one who does mimic, can't login unless he knows the key.
    • In addition, GDM can be voice enabled to speak, Password correct, Incorrect, Invalid voice etc..

    People

    Srinivasa Ragavan originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
