    • VYM export to Tomboy

    Tags: Idea, Mindmap, vym, Tomboy, openSUSE, Desktop

    Description

    vym is a mindmapping application, which allows to effectively collect and sort ideas, link to documents and edit text. There are various export filters already for OpenOffice, HTML, LaTex, etc. But so far vym lacks a filter to export to Tomboy, a kind of personal WIKI.

    Implementation could be done in Qt or xslt.

    People

    Uwe Drechsel originated this idea. No one is currently working on implementing this idea (it's up for grabs!). Uwe is going to help with implementation.
