    • Web based linux installer

    Tags: Idea, web20, yast, management

    Description

    Internet availability and bandwidth is constantly increasing. Opensuse 10.3 wants to take advantage of that and make online repositories and packages available early during installation.

    Pushing this a bit further could also make translations, help, patches, etc. available early in the installation workflow.

    As projects already tackle the problem of a web UI for YaST, it should be possible to run the complete installation workflow in a browser window.

    Moving Ruby on Rails framework make this idea even more feasible.

    Memory and usability constraints are probably most limiting here. But we'll only know for sure //after// implementing a prototype.

    People

    Klaus Kämpf originated this idea. No one is currently working on implementing this idea (it's up for grabs!).

    Dirk Müller has khtml experience and knows how to run a //stripped down// browser window.

    Siegfried Olschner could advise on the usability.

    Nat Friedman might point us to an experienced web-designer.
