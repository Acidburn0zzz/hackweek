    • write a fuzzer for php

    Tags: Idea, Security, QA, LAMP

    Description

    Write a program that tries to find potential security bugs in the PHP engine by feeding it with slightly invalid code. If php crashes instead of aborting with an error, we have a candidate. Save the testcase and repeat.

    People

    Cristian Rodriguez and Michal Marek originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
