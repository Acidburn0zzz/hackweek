    • Write a library for lomoco and fully support MX/VX Revolution and G series Logitech mice

    Tags: InProgress, HW-Hacker, Mobile, Desktop

    Description

    Find out the new protocols which are used by the MX/VX Revolution and the G series. Rewrite lomoco and start with a library.

    lomoco is a tool configure vendor-specific options on Logitech USB mice.

    Status

    I've discovered how to read the battery status of the MX/VX Revolution and VX Nano.

    http://www.lomoco.org/lomoco_battery.c

    Reconnection utility to pair mouse and receiver:

    http://www.lomoco.org/lomoco_reconnect.c

    G series resolution switching code:

    http://www.lomoco.org/g_hack.c

    I wasn't able to figure out how Logitech identifies the cordless mouse connected to the receiver. I will start to write the library in the next weeks.
