    • Write a new filesystem with FUSE

    Tags: Idea, Kernel, Filesystem, FUSE

    Description

    FUSE allows you to write a custom filesystem in userspace using almost any language that you are comfortable with. Examples of this are sshfs, ftpfs and even a beagle-based filesystem. Use the FUSE interface to create the custom filesystem that you always dreamed of.

    Information on using FUSE can be found at http://fuse.sourceforge.net/

    Heck, you could even implement ZFS in FUSE to get an idea of how well it works and to handle the license issues, a group that has started doing this can be found at http://www.wizy.org/wiki/ZFS_on_FUSE
