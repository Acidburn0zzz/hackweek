    • YaST as a WebService

    Tags: InBuildService, Done, yast, Management, WebService, JudgeMe

    Description

    Making YaST functionality (remotely) available, esp. to other (management) tools, is a long-standing task. Previous approaches ranged from writing a YaST daemon to modelling YaST within CIM and expose it through the CIMOM. All these needed a lot of time and experience.

    WebService-Management (ws-man, see Openwsman) now makes it possible to transport remote function calls without the all the burden mentioned above.
