    • YAST Backup Module

    Topic tags: Server, IdeaPool Product tags: openSUSE, SLES, SLED Status tags: Idea Help tags:

    Tags: Idea, Yast, rsnapshot

    Description

    The idea is to create a YAST module which configurate rsnapshot (http://www.rsnapshot.org/).

    requirements on the module: * set up cronjob to start rsnapshot to predefined time * local and remote backup sources * different backup protocolls like rsync, rsync over ssh, local ... * the possibility to take a look on the actuall used disk size

    People

    Michael Skibbe originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
