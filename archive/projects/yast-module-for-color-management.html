    • yast module for color management

    Tags: Idea, SLED, Graphics, Desktop, YaST

    Description

    With more and more color management aware applications it would be nice to have a GUI available to set color profiles for display(s), printer(s) and other devices.

    Yast seems to be ideal place for this.

    People

    Radek Doulik originated this idea. No one is currently working on implementing this idea (it's up for grabs!).
